These are 7 types of operators in Python

Arithmatic Operator : It uses to performs math matical operations

example: Addition (+), Subtraction(-), multiplication(*), Division(/)

Assignment Operator : It uses to assign values to variables

example: =, +=, -=, *=, /=

Comparison Operator : It uses to compare values

example: ==, !=, >, <, >=, <=

Logical Operator : It uses to combine conditional statements

example: and, or, not

Identity Operator : It uses to check if both variables are the same object

example: is, is not

Membership Operator : It uses to check if a value is present in a sequence

example: in, not in

Bitwise Operator : It uses to perform operations on bits

example: & , |, ^, ~, <<, >>
